Proyecto Final – Juego Adivina el Número
Objetivo del proyecto

El objetivo de este proyecto es desarrollar un software funcional de un juego sencillo utilizando Python, con el fin de reforzar los conocimientos básicos de lógica de programación, uso de estructuras de control, funciones y estructuras de datos, además de aprender a crear y administrar un repositorio en GitHub.
A través de este proyecto se busca que el estudiante comprenda mejor el funcionamiento del lenguaje Python y el uso de GitHub como herramienta para el control de versiones y la entrega de proyectos.

  Alcance del proyecto

El proyecto consiste en la implementación de un juego llamado “Adivina el número”, en el cual el programa genera un número aleatorio entre 1 y 20 y el usuario dispone de 5 intentos para adivinarlo.
El juego proporciona retroalimentación en cada intento indicando si el número ingresado es mayor o menor que el número secreto, y finaliza cuando el usuario acierta o se agotan los intentos.
El alcance del proyecto incluye:

Generación de números aleatorios.

Interacción con el usuario mediante consola.

Uso de funciones para organizar el código.

Aplicación de estructuras de control como ciclos y condicionales.

Publicación del proyecto completo en un repositorio de GitHub.
